app_helpers
===========

Sets up a fresh Rails project and adds useful helpers. 


Set up a fresh project
----------------------

### Create empty Rails project

	rails my_app

### Install

	script/plugin install git://github.com:winton/app_helpers.git

### Run rake task

	rake app_helpers

### Update plugins (see **config/plugins.rb**)

	rake plugins:update
